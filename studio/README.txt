HOW TO COMPILE (on Windows 7)
=============================
1 install MSYS2 if you haven't already
2 open MSYS and install 'gtkmm3', 'toolchain', and 'make' via pacman
3 cd to this path
4 then type: make


HOW TO COMPILE (on GNU/Linux)
=============================
1 install 'gtkmm3', 'toolchain', and 'make'
3 cd to this path
4 type: 'make'
