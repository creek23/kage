HOW TO COMPILE BasicXML (on Windows 7)
======================================
1 install MSYS2 if you haven't already
2 open MSYS and install 'toolchain', and 'make' via pacman
3 cd to this path
4 then type: 'make'


HOW TO COMPILE BasicXML (on GNU/Linux)
======================================
1 open Terminal and install 'g++'
2 cd to this path
3 type: 'make'
